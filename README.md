# Node pokemondb
Web Scraper in Nodejs, with Puppeteer

# Table of Contents

* [Version](#version)
* [Quick start](#quick-start)


# Version
- Node 10
- Puppeteer 1.18.x

# Quick start

- Clone following repo:  
```javascript
git clone https://github.com/JhonatanMedeiros/node-pokemondb.git
```  
- Run `yarn install`
- Run `yarn start`
